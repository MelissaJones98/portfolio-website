The Witch's Book Nook (Book Club Website)

Description

As part of a Codecademy course I was tasked to create a website for a club, I chose to create a website for a fictional book club, showcasing skills in responsive web design.

Features

The website contains:

  • a home page that greets the user and introduces the book club
  • a brief history of the club creation and details of the founder
  • details about upcoming and previous books that the club has read
  • a schedule of the upcoming club meetings as well as details of the meeting location
  • a sign up page for people to become members of the book club

How to Use

Run the index.html file in the browser and navigate the site using the buttons on the nav bar.

Technologies

  • HTML
  • CSS

Magical Message Generator

Description

Codecademy Portfolio Project. Message generator program. Every time a user runs the program they will get a new randomised output. Upon running the program a unique message will be generated comprised of three different components selected at random from three different arrays.

Features

  • A basic HTML webpage styled using CSS that provides a visual for the message generator program written using JavaScript
  • Functions which are called using onclick on two different buttons (generate and reset) to generate the messages and reset
  • The messages are compiled at random by the functions selecting items from three different arrays and concatenating them

How to Use

Option 1: Node

node ./script.js

Option 2: HTML

Run index.html in the browser

Technologies

  • HTML
  • CSS
  • JavaScript
