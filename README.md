# Portfolio Website (A Codecademy Project)

## Description
I built a personal portfolio website. It is a static website where I share my portfolio projects. I planned the project, created a wireframe, and set up a GitHub repository for it. 
The website includes two pages: the main page about me and my projects and skills and a contact page which contains my contact information should employers wish to reach me. 
I started by just creating the HTML files and building the skeleton structure of the website. Next, I styled it using CSS and taking into consideration constrasting colours for optimal readability. 
I ensured that my styling was responsive and considered three viewport types: mobile, tablet and desktop. The styling for each viewport ensure ease of viewing and ensures all content is visible and accessible
to all users. I made sure to commit after each significant change I made to ensure I had a fleshed out project history and version control using Git/GitHub. Finally, I included one interactive element - 
a button that reveals my skills list and disappears when the user clicks it. I did this using JavaScript.

## Features
- Main page containing three sections: About Me, Projects and Skills
- Nav bar uses links within the same page as well as linking to other pages
- Each project has a GitHub logo next to its title which when clicked by the user will take them to the GitHub repo for that project
- Interactive button within skills section
- All buttons and clickable items show clickability using the hover pseudo selector in CSS
- Contact page containing contact information

## How to Use
Run the index.html file in the browser.
